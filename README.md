# English B2 Master

Lokalna platforma web do nauki angielskiego na poziomie B2.

## Co zawiera
- test poziomujący B2
- practice mode z filtrowaniem po obszarach
- symulator egzaminu na czas
- tłumaczenia PL → EN
- transformacje zdań
- reading comprehension
- zapis postępów w localStorage
- bank błędów i statystyki

## Jak uruchomić
1. Otwórz plik `index.html` w przeglądarce.
2. Dla pełnego komfortu możesz też uruchomić prosty serwer lokalny, np.:
   - `python -m http.server`
3. Następnie wejdź na adres lokalny pokazany przez serwer.

## Struktura
- `index.html` — layout aplikacji
- `styles.css` — wygląd UI
- `data.js` — baza 171 zadań B2
- `app.js` — logika platformy
